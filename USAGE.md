# Usage on running in dev mode

Requirements are 
- node 
- live server OR python

## Starting the web server
Either install live server as mentioned in the README.md or run
`python -m http.server 5500`
and i believe this should work

## Starting the node backend
1. Run `npm i` in this folder
2. Run `node api-server/server.js`

The frontend should be on port 5500 and backend on 5501
